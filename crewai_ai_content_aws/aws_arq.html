En nuestra web awsdesdecero.com nos encontramos con la necesidad de mejorar la eficiencia y calidad de nuestros productos. Para lograr esto, hemos implementado una arquitectura de AWS que utiliza visión artificial industrial para detectar defectos en nuestros productos.

Introducción

La inspección manual es un proceso lento y costoso que puede llevar hasta 10 años para encontrar todos los defectos. El uso de la visión artificial industrial puede acelerar este proceso y reducir los costes.

Servicios clave de AWS utilizados

Para implementar esta arquitectura, hemos utilizado varios servicios de AWS:

  • Tosca: Es un servicio que permite crear modelos de aprendizaje automático utilizando los datos del producto y la inspección manual.
  • Computer Vision: Es un conjunto de APIs diseñadas para ayudar a desarrolladores a analizar imágenes y videos y realizar tareas como el reconocimiento de objetos, la detección de defectos, etc.
  • Lambda: Es un servicio que permite ejecutar funciones escalables y serverless en respuesta a eventos específicos. En este caso, se utiliza para procesar las imágenes de los productos y enviarlas al modelo de aprendizaje automático.
  • Amazon S3: Se utiliza para almacenar las imágenes de los productos y las respuestas del modelo de aprendizaje automático.

La arquitectura consta de los siguientes componentes:

  • Un servidor Lambda que procesa las imágenes de los productos y envíalas al modelo de aprendizaje automático.
  • Un modelo de aprendizaje automático creado con Tosca, que analiza las imágenes y detecta defectos.
  • La inspección manual, que proporciona datos de calidad para el modelo de aprendizaje automático.
  • Amazon S3, que almacena las imágenes de los productos y las respuestas del modelo de aprendizaje automático.

En conclusión, la arquitectura de AWS utilizada en nuestra web awsdesdecero.com permite detectar defectos en nuestros productos utilizando visión artificial industrial. La combinación de servicios clave de AWS como Tosca, Computer Vision, Lambda y Amazon S3 ha permitido automatizar el proceso de inspección manual y reducir los costes. Nuestra experiencia con esta arquitectura nos ha demostrado la eficiencia y calidad de nuestros productos.
